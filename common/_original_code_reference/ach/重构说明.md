✅ **路径动态化**：使用 `Path(__file__).parent.absolute()` 动态确定脚本目录

## 主要改动

### 1. 路径处理改进

**原代码问题：**

```python
# 硬编码相对路径
STORAGE_STATE_FILE = "storage_state.json"
CONFIG_FILE = "ach_config.json"
```

**重构后：**
```python
# 动态获取脚本所在目录
from pathlib import Path
SCRIPT_DIR = Path(__file__).parent.absolute()
STORAGE_STATE_FILE = str(SCRIPT_DIR / "storage_state.json")
CONFIG_FILE = str(SCRIPT_DIR / "savexiumi_config.json")
```

### 2. 函数参数增强

#### `load_config()` 函数
```python
# 原函数签名
def load_config(config_path):

# 新函数签名  
def load_config(config_path=None):
    """加载配置文件
    Args:
        config_path (str, optional): 配置文件路径，默认使用脚本目录下的config.json
    """
```

#### `init_browser()` 函数
```python
# 原函数签名
def init_browser(playwright_instance, headless=False):

# 新函数签名
def init_browser(playwright_instance, headless=False, storage_state_path=None):
    """初始化浏览器和上下文
    Args:
        playwright_instance: Playwright实例
        headless (bool): 是否无头模式
        storage_state_path (str, optional): 存储状态文件路径，默认使用脚本目录下的storage_state.json
    """
```

#### `main()` 函数
```python
# 原函数签名
def main(manual_login=False):

# 新函数签名
def main(manual_login=False, config_path=None, storage_state_path=None):
    """主程序入口
    Args:
        manual_login (bool): 是否手动登录
        config_path (str, optional): 配置文件路径
        storage_state_path (str, optional): 存储状态文件路径
    """
```

## 使用方式

### 1. 直接运行（推荐）
```bash
# 在 ach.py 所在目录下直接运行
python ach.py
```

### 2. 作为模块导入使用
```python
from ach import main

# 使用默认配置文件和存储状态文件
main()

# 或者指定自定义路径
main(
    config_path="/自定义路径/savexiumi_config.json",
    storage_state_path="/自定义路径/storage_state.json"
)
```

### 3. 从其他目录调用
```python

from pathlib import Path

# 添加 ach.py 所在目录到 Python 路径
ach_dir = Path("/path/to/ach/directory")
sys.path.insert(0, str(ach_dir))

from ach import main
main()
```

# 字典调用方法

## 概述

本文档详细说明如何通过**外部 JSON 配置文件**的方式，将 `LOOP_CONFIGS` 字典保存到外部文件，然后让脚本读取该配置并运行。

---

## 核心概念

这种调用方式的核心思想是：
1. 在外部创建一个 **JSON 配置文件**，定义任务列表和账号配置
2. 配置脚本中的 `EXTERNAL_CONFIG_FILE` 常量指向该 JSON 文件
3. 调用 `main()` 函数，脚本会自动读取外部配置并执行任务

---

## JSON 配置文件格式

### 完整配置模板

```json
{
    "account_index": 0,
    "loop_configs": [
        {
            "prompt": "黄昏的场景，天空中有一只会飞的翼龙剪影",
            "save_name_template": "大理洱海游船",
            "aspect_ratio": "1:1",
            "upload_image_path": "C:\\Users\\Administrator\\Desktop\\公众号文章\\output\\大理洱海游船1.jpg",
            "old_file_name": "大理洱海游船1",
            "file_extension": "jpg",
            "download_dir": "C:\\Users\\Administrator\\Desktop\\公众号文章\\output"
        },
        {
            "prompt": "古镇吊脚楼，传统建筑特色",
            "save_name_template": "古镇吊脚楼",
            "aspect_ratio": "1:1",
            "upload_image_path": "C:\\Users\\Administrator\\Desktop\\公众号文章\\output\\古镇吊脚楼1.jpg",
            "old_file_name": "古镇吊脚楼1",
            "file_extension": "jpg",
            "download_dir": "C:\\Users\\Administrator\\Desktop\\公众号文章\\output"
        }
    ]
}
```

### 字段说明

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| `account_index` | `int` | 否 | 账号索引，从 0 开始。为空时使用脚本中 `ACCOUNT_INDEX` 常量 | `0` |
| `loop_configs` | `array` | **是** | 任务配置列表 | 见下方详解 |

### loop_configs 数组字段

每个任务配置是一个对象，支持以下字段：

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| `prompt` | `string` | 是 | AI 生成图片的提示词 | `"黄昏的场景，天空中有一只会飞的翼龙"` |
| `save_name_template` | `string` | 是 | 保存文件名模板 | `"result_{index}"` |
| `aspect_ratio` | `string` | 否 | 图片比例 | `"1:1"`, `"16:9"`, `"9:16"` |
| `upload_image_path` | `string` | 否 | 参考图片路径 | `"C:\\Users\\Desktop\\1.jpg"` |
| `old_file_name` | `string` | 否 | 旧文件名 | `"图片1"` |
| `file_extension` | `string` | 否 | 输出图片格式 | `"jpg"`, `"png"`, `"bmp"` |
| `download_dir` | `string` | 否 | 下载保存目录 | `"C:\\Users\\Desktop\\output"` |

### save_name_template 可用变量

- `{index}` - 任务索引，从 1 开始
- `{prompt}` - 提示词内容（仅保留前 20 个字符）
- `{aspect_ratio}` - 图片比例

### 步骤 1：创建 JSON 配置文件

```json

            "old_file_name": "大理洱海游船1",
            "file_extension": "jpg",
            "download_dir": "C:\\Users\\Administrator\\Desktop\\公众号文章\\output"
        },
        {
            "prompt": "古镇吊脚楼，传统建筑特色",
            "save_name_template": "古镇吊脚楼_{index}",
            "aspect_ratio": "1:1",
            "upload_image_path": "C:\\Users\\Administrator\\Desktop\\公众号文章\\output\\古镇吊脚楼1.jpg",
            "old_file_name": "古镇吊脚楼1",
            "file_extension": "jpg",
            "download_dir": "C:\\Users\\Administrator\\Desktop\\公众号文章\\output"
        }
    ]

```

**建议存放位置**：`C:\Users\Administrator\Desktop\OpsDeck\config\jimeng_config.json`

### 步骤 2：配置脚本常量

修改 `playwright_template.py` 中的 `EXTERNAL_CONFIG_FILE` 常量：

```python
# 外部配置文件路径（用于字典调用方法）
EXTERNAL_CONFIG_FILE = r"C:\Users\Administrator\Desktop\OpsDeck\config\jimeng_config.json"
```

### 步骤 3：运行脚本

直接运行脚本即可：

```python
import sys
sys.path.append(r"C:\Users\Administrator\Desktop\OpsDeck\src")

from opsdeck.common.jimeng.playwright_template import main

# 无需传参，脚本会自动读取外部配置文件
main()
```

## 高级用法

### 方法 1：通过常量指定配置文件（推荐）

在脚本中设置 `EXTERNAL_CONFIG_FILE` 常量：

```python
# playwright_template.py 中
EXTERNAL_CONFIG_FILE = r"C:\Users\Administrator\Desktop\OpsDeck\config\jimeng_config.json"
```

然后直接调用：

```python
from opsdeck.common.jimeng.playwright_template import main
main()
```

### 方法 2：通过参数指定配置文件

在调用时传入配置文件路径：

```python
import sys
sys.path.append(r"C:\Users\Administrator\Desktop\OpsDeck\src")

from opsdeck.common.jimeng.playwright_template import main

# 运行时指定外部配置文件
main(external_config_file=r"C:\Users\Administrator\Desktop\OpsDeck\config\jimeng_config.json")
```

### 方法 3：JSON 中只定义任务，不指定账号

如果 JSON 文件中不指定 `account_index`，会使用脚本中的 `ACCOUNT_INDEX` 常量：

```json
{
    "loop_configs": [
        {
            "prompt": "任务1",
            "save_name_template": "result_{index}",
            "aspect_ratio": "1:1"
        }
    ]
}
```

### 方法 4：部分配置使用外部，部分使用代码

外部 JSON 可以只定义 `loop_configs`，同时在代码中指定 `account_index`：

**JSON 文件** (`config.json`):
```json
{
    "loop_configs": [
        {
            "prompt": "任务1",
            "save_name_template": "result_{index}"
        }
    ]
}
```

**Python 调用**:
```python
from opsdeck.common.jimeng.playwright_template import main

# 外部配置提供 loop_configs，代码指定账号
main(account_index=1, external_config_file=r"C:\config.json")
```

## 脚本修改说明

为了支持外部字典调用方式，脚本进行了以下修改：

### 1. 新增常量

```python
# 设置为外部 JSON 配置文件的绝对路径，为 None 时不使用外部配置

```

### 2. 新增函数

```python
def load_external_config(config_file: str) -> dict:
    """从外部 JSON 文件加载配置"""
    import json
    # 读取并解析 JSON 文件
    # 返回包含 account_index 和 loop_configs 的字典
```

### 3. 修改 main 函数签名

```python
def main(account_index: Optional[int] = None, loop_configs: list = None, external_config_file: str = None) -> None:
```

新增参数：
- `external_config_file`: 外部配置文件路径，为 None 时使用常量 `EXTERNAL_CONFIG_FILE`

---

## 注意事项

1. **JSON 格式**：确保 JSON 文件格式正确，可以使用在线 JSON 校验工具验证
2. **路径格式**：Windows 路径建议使用双反斜杠 `\\` 或原始字符串
3. **编码**：JSON 文件建议保存为 UTF-8 编码
4. **优先级**：
   - 直接传参 > 外部配置文件 > 脚本常量
   - 例如：同时设置了 `account_index=1` 和外部配置文件中的 `account_index: 0`，最终使用 `1`
5. **错误处理**：如果配置文件不存在或格式错误，会回退到使用脚本常量

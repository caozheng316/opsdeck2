# 直接调用方法

## 概述

本文档详细说明如何通过**外部 Python 代码调用**的方式，直接传参运行 `playwright_template.py` 脚本的主函数。

---

## 调用方式

### 基本语法

```python
import sys
sys.path.append(r"C:\Users\Administrator\Desktop\OpsDeck\src")

from opsdeck.common.jimeng.playwright_template import main

# 调用主函数
main()
```

---

## 参数说明

### main() 函数签名

```python
def main(account_index: Optional[int] = None, loop_configs: list = None) -> None:
```

| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| `account_index` | `int` | 否 | `ACCOUNT_INDEX` 常量值 | 账号索引，从 0 开始 |
| `loop_configs` | `list` | 否 | `LOOP_CONFIGS` 常量值 | 循环任务配置列表 |

---

## 调用示例


## loop_configs 字典字段详解

每个任务配置是一个字典，支持以下字段：

| 字段 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| `prompt` | `str` | 是 | AI 生成图片的提示词 | `"黄昏的场景，天空中有一只会飞的翼龙"` |
| `save_name_template` | `str` | 是 | 保存文件名模板 | `"result_{index}"` |
| `aspect_ratio` | `str` | 否 | 图片比例 | `"1:1"`, `"16:9"`, `"9:16"` |
| `upload_image_path` | `str` | 否 | 参考图片路径（需要 AI 参考此图生成） | `r"C:\Users\Desktop\1.jpg"` |
| `old_file_name` | `str` | 否 | 旧文件名（用于识别旧图片） | `"图片1"` |
| `file_extension` | `str` | 否 | 输出图片格式 | `"jpg"`, `"png"`, `"bmp"` |
| `download_dir` | `str` | 否 | 下载保存目录 | `r"C:\Users\Desktop\output"` |

### save_name_template 可用变量

- `{index}` - 任务索引，从 1 开始
- `{prompt}` - 提示词内容（仅保留前 20 个字符）
- `{aspect_ratio}` - 图片比例

**示例**：
- `"result_{index}"` → `result_1`, `result_2`, ...
- `"result_{index}_{prompt}_{aspect_ratio}"` → `result_1_黄昏的场景_1:1`

## 注意事项

1. **路径格式**：Windows 路径建议使用原始字符串 `r"C:\path\to\file"`，避免转义问题
2. **账号索引**：`account_index` 从 0 开始，0 = 第一个账号，1 = 第二个账号，以此类推
3. **登录状态**：首次运行需要手动登录，登录状态会保存在用户数据目录中
4. **错误处理**：如果任务执行失败，脚本会记录错误日志并继续执行下一个任务
5. **浏览器关闭**：默认会等待用户按回车键后关闭浏览器，设置 `WAIT_FOR_INPUT = False` 可自动关闭
